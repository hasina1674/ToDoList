## ✅ Simple To-Do List Web App

A lightweight and interactive **To-Do List** web application built using **HTML**, **CSS**, and **JavaScript**. This app helps users manage their daily tasks by allowing them to add, complete, and delete tasks easily.

---

### 🚀 Features

* 📝 Add tasks
* ✅ Mark tasks as completed (toggle by clicking)
* 🗑️ Delete individual tasks
* 💡 Simple and intuitive UI
* ⚡ Fast and responsive (no frameworks used)

---

### 🛠️ Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript (ES6)


---

### 💻 How to Run Locally

1. **Clone the repository:**

```bash
git clone https://github.com/hasina1674/ToDoList-Web.git
cd ToDoList-Web
```

2. **Open the app:**

Double-click the `index.html` file or open it in your browser.

---

### 🧠 How It Works

* User types a task in the input box and clicks **Add**.
* The task appears in the list.
* Clicking a task toggles its **completed** state.
* Clicking **Delete** removes the task from the list.
